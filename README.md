# EthosiFi — Proof of Ethics, On-Chain

EthosiFi is building the immune system for Web4 — a behavioral trust layer that verifies project integrity before launch through on-chain reputation scoring.

## Features

- ✅ Real-time Gnosis Safe treasury verification
- ✅ LP lock detection on Base Mainnet
- ✅ Team transparency via ENS/SIWE
- ✅ Fair launch pledge enforcement
- ✅ No KYC, no documents — just on-chain proof

## Tech Stack

- **Frontend**: Next.js 14, React 18
- **Blockchain**: Ethers.js 5.7
- **RPC**: Ankr Base (CORS-enabled)
- **Hosting**: Vercel
